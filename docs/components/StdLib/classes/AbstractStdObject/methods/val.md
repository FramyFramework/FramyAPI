# val
## Syntax

```php
public function val($value = null);
```

## Remarks

Return, or update, current standard objects value.

## Arguments

| DataType | Name | Description |
| --- | --- | --- |
|  | $value | If $value is set, value is updated and ArrayObject is returned. |

## Returns

| DataType | Name | Description |
| --- | --- | --- |
| mixed | | |
