# __set
## Syntax

```php
public function __set($name, $value);
```

## Remarks

Set internal data as if it was a real object

## Arguments

| DataType | Name | Description |
| --- | --- | --- |
| String | $name| |
| Mixed | $value | |

| --- | --- | --- |
| Void | | |
