# get
## Syntax

```php
public function get($name, $default = null);
```

## Remarks

Get value or return $default if there is no element set.

## Arguments

| DataType | Name | Description |
| --- | --- | --- |
| String | $name| |
| Mixed | $default| |

## Returns

| DataType | Name | Description |
| --- | --- | --- |
| mixed | | Config value or default value |
