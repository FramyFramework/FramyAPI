# offsetExists
## Syntax

```php
public function offsetExists($offset);
```

## Remarks

(PHP 5 &gt;= 5.0.0)<br/>
Whether a offset exists

## Arguments

| DataType | Name | Description |
| --- | --- | --- |
| Mixed | $offset | An offset to check for. | 

## Returns

| DataType | Name | Description |
| --- | --- | --- |
| Boolean | | true on success or false on failure. |
