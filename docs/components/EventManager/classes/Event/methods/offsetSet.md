# offsetSet
## Syntax

```php
public function offsetSet($offset, $value);
```

## Remarks

(PHP 5 &gt;= 5.0.0)<br/>
Offset to set

## Arguments

| DataType | Name | Description |
| --- | --- | --- |
| Mixed | $offset | The offset to assign the value to. | 
| Mixed | $value | The value to set. |

## Returns

| DataType | Name | Description |
| --- | --- | --- |
| void | | |
